Cada linha da tabela representa um registro de um paciente. Temos 15 variáveis:¶

  • paciente_idade: Float, representando a idade do paciente;
  • paciente_enumsexobiologico: Objeto string, representando o sexo biolgico do paciente (M=masculino, F=feminino, I=indefinido);
  • paciente_racacor_valor: Objeto string, descrevendo a raa/cor do paciente;
  • paciente_endereco_coibgemunicipio: Float, representando o cdigo do municpio do paciente segundo o IBGE;
  • paciente_endereco_nmmunicipio: Objeto string, descrevendo o nome do municpio do paciente segundo o IBGE;
  • paciente_endereco_uf: Objeto string, descrevendo o nome do estado do paciente segundo o IBGE;
  • estalecimento_nofantasia: Objeto string, descrevendo o nome do estabelecimento no qual o paciente foi vacinado;
  • estabelecimento_municipio_nome: Objeto string, descrevendo o nome do municpio do estabelecimento no qual o paciente foi vacinado;
  • estabelecimento_uf: Objeto string, descrevendo o nome do estado do estabelecimento no qual o paciente foi vacinado;
  • vacina_grupoatendimento_nome: Objeto string, descrevendo o grupo de atendimento em que o paciente se enquadrou para a vacinao;
  • vacina_categoria_nome: Objeto string, descrevendo a categoria em que o paciente se enquadrou para a vacinao;
  • vacina_fabricante_nome: Objeto string, descrevendo o nome do fabricante da vacina que o paciente tomou;
  • vacina_dataaplicacao: Objeto string, indicando a data em que o paciente foi vacinado;
  • vacina_descricao_dose: Objeto string, indicando a dose correspondente que o paciente tomou;
  • vacina_nome: Objeto string, descrevendo o nome da vacina que o paciente tomou;

Quais os tipos das variáveis?¶

Sobre as variveis numricas, paciente_idade uma varivel racional e vacina_dataaplicacao intervalar, todas as outras so variveis categricas. Para garantir a integridade do tipo, vamos converter a varivel vacina_dataaplicacao de string para datetime. Alm disso, para possibilitar anlises mais profundas, vamos criar uma varivel temporria chamada vacina_dias_desde_aplicacao, que representa a quantidade de dias desde a aplicao da vacina at 14/05/2021, data em que os dados foram coletados.

df.loc[:,'vacina_dataaplicacao'] = df.loc[:,'vacina_dataaplicacao'].astype('datetime64[ns]')
In [211]:
df['vacina_dias_desde_aplicacao'] = (pd.to_datetime('2021-05-14') - df['vacina_dataaplicacao']).dt.days

# Exibindo valores dos limites inferiores e superiores de cada varivel numrica
def get_box_plot_data(labels, bp):
    rows_list = []

    for i in range(len(labels)):
        dict1 = {}
        dict1['Varivel'] = labels[i]
        dict1['Limite inferior'] = bp['whiskers'][i*2].get_ydata()[1]
        dict1['Q1'] = bp['boxes'][i].get_ydata()[1]
        dict1['Q2 - Mediana'] = bp['medians'][i].get_ydata()[1]
        dict1['Q3'] = bp['boxes'][i].get_ydata()[2]
        dict1['Limite superior'] = bp['whiskers'][(i*2)+1].get_ydata()[1]
        rows_list.append(dict1)

    return pd.DataFrame(rows_list)
    
bp = dados_numer.boxplot(['paciente_idade', 'vacina_dias_desde_aplicacao'], return_type='dict', widths=0)
